# CYBER RHYTHMへ進化！デザイン一新＆隠し難易度実装

ご要望の通り、これまでの「可愛いパステル調」から雰囲気を一転させ、本格的な音ゲーらしい**サイバーパンク風デザイン**への全面リニューアルが完了しました！

## ✨ 今回の追加・変更点

### 1. 🌃 UIとデザインの完全サイバー化
- **ホーム・ポーズ・リザルト画面**: プレイ画面に合わせて、すべての画面の背景をダークカラー＆サイバーグリッド（網目）の世界観に統一しました。
- **テキストと絵文字の排除**: 以前まで使用していた絵文字（🏠や🎵など）をすべて削除し、`[ INITIALIZE ]` `[ RESUME ]` `[ MISSION CLEARED ]` のような、システム的・サイバーパンク的なテキストデザインに変更しました。
- **判定ラインの変更**: 判定エリアの中心にあった「丸」を無くし、**画面を横に貫く一本のレーザーライン（判定線）**に変更しました。より直感的で音ゲーらしい見た目になっています。

### 2. 🎧 追加デフォルト楽曲の設定
- ご自身で `game` フォルダに配置していただいた `127_BPM155.mp3` や `420_BPM108.mp3` などの楽曲ファイルを検知し、ホーム画面の**「デフォルト曲から選ぶ（SELECT DEFAULT TRACK）」のセレクトボックス**にすべて追加しました！
- これにより、ファイルを選択しなくてもリストから選ぶだけですぐに遊べるようになっています。

### 3. 🎧 サイバー風のヒット音（電子音）
ノーツを叩いた時の音を、全体の世界観に合わせて「シンセサイザー風の電子音（レーザーのようなピュン！という音）」に変更しました。
- **PERFECT**: 高くてキレのあるレーザー音
- **GREAT**: 少しマイルドな電子音
- **MISS**: 低くダウンするノイズ音

### 4. 🔥 隠しコマンド：超絶難易度「HELL」
- スタート画面（曲や難易度を選ぶ画面）を開いた状態で、キーボードで **「H」「E」「L」「L」** と順番にタイピングしてみてください！
- 画面が赤くフラッシュし、選択肢に赤く光る **「HELL」** モードが出現します。
- **HELLの仕様**: スピードは爆速（12）、ノーツ間隔は脅威の0.05秒。曲のほぼすべての波形ピークを容赦なく拾う鬼畜仕様です。

## 📝 遊び方・ご確認
1. ブラウザをリロードして、UI全体から絵文字が消え、クールなテキストに統一されていることを確認してください。
2. セレクトボックスから、ご自身で追加された楽曲（例：Track 127）を選んでプレイしてみてください！
