# サイバー風デザインへの全面リニューアルと難易度の再定義

現在の「パステル・サンリオ風」から大きく舵を切り、本格的な音ゲーらしい「サイバー感（ネオン・ダーク・未来的）」のあるデザインへの一新と、難易度・隠し要素（HELL）の実装を行います。

## User Review Required
> [!IMPORTANT]
> **デザインの方向性と隠しコマンドについて**
> 1. **デザイン一新**: これまで画面中央にいた「キャラクター」を完全に削除し、背景をダーク（黒・深海色など）にして、ノーツやUIがネオン（シアンやマゼンタなど）で光る「サイバーパンク風」のデザインに書き換えます。よろしいでしょうか？
> 2. **隠しコマンド「HELL」の仕様**: スタート画面でキーボードで直接 **「H」「E」「L」「L」** と順番にタイピングする（スマホの場合は特定のボタンを長押し、またはロゴを10回タップするなど）と、画面が赤く光って「HELL」難易度が出現する、というギミックはいかがでしょうか？

## Proposed Changes

### 1. サイバー感のあるUI・背景への大改修（HTML / CSS）
- **キャラクターの削除**: HTMLから `<div class="character">` 関連の要素をすべて削除し、CSSのアニメーションも破棄します。
- **カラーパレットの変更**:
  - 背景色: ダークグレー〜黒（`#111` など）
  - アクセントカラー（ネオン）: シアン（水色系）、マゼンタ（ピンク紫系）、ネオングリーン
- **UIスタイリング**: 丸みを帯びたデザインから、シャープで直線的なデザインに変更。各要素に `box-shadow` を用いた「発光（グロウ）エフェクト」を適用し、未来的なサイバー空間を表現します。

### 2. 難易度の再定義と「HELL」の追加（JS）
- **EASY（子供向け・初心者向け）**:
  - ノーツ速度: かなり遅い
  - 最小間隔（密度）: 0.8秒（同時にたくさん降ってこない、単発メイン）
- **NORMAL（一般向け）**:
  - ノーツ速度: 標準的
  - 最小間隔: 0.3秒（裏拍などのリズムも適度に拾う）
- **HARD（音ゲーマー向け）**:
  - ノーツ速度: 速い
  - 最小間隔: 0.12秒（細かい音もすべて拾い、連打も要求される）
- **HELL（超上級者向け・隠し）**:
  - ノーツ速度: 爆速（見切るのが難しいレベル）
  - 最小間隔: 0.05秒（曲のほぼすべての波形ピークをノーツ化）

### 3. 隠しコマンド機能の実装（JS）
- スタート画面において、特定のキーボード入力（例：`HELL`）や、特定のタップ操作が行われたことを検知するロジックを追加。
- コマンド成功時にサイバーな警告音（または演出）とともに、難易度選択UIに赤く光る「HELL」オプションを出現させます。
